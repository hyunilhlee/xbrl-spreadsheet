XBRL/XML 파일을 여기에 놓으세요

사용 방법과 지원 수식

선택한 XBRL과 작업 데이터는 외부 서버로 업로드하지 않고 브라우저 안에서만 처리됩니다. 앱은 정적 호스팅에서 index.html과 ./src/spreadsheet-core.mjs를 함께 불러오므로 로컬에서는 HTTP 서버로 실행하세요.

기본 사용

XBRL 열기에서 인스턴스 파일과 *_lab.xml 라벨 파일을 함께 선택하면 재무표, Facts, Contexts, 계산 시트가 생성됩니다. 셀을 더블클릭하거나 선택 후 바로 입력하면 편집할 수 있습니다.

셀을 선택해 수식 만들기

셀이나 수식 입력줄에서 =로 편집을 시작한 뒤 그리드의 셀을 클릭하거나 범위를 드래그하면 현재 커서 위치에 A1 참조가 입력됩니다. 참조에 커서를 두고 F4를 누르면 A1 → $A$1 → A$1 → $A1 순서로 고정 방식이 바뀝니다.

채우기 핸들

셀 또는 범위 오른쪽 아래의 파란 핸들을 위·아래·왼쪽·오른쪽으로 드래그하세요. 한 값은 반복하고, 일정 간격의 숫자·ISO 날짜·문자+숫자 값은 계열을 이어갑니다. 수식은 상대 참조를 이동하고 원본 서식을 복사합니다. 드래그 중 Esc로 취소하거나 적용 후 Ctrl+Z/Cmd+Z로 채우기 전체를 되돌릴 수 있습니다.

수식 예시

용도입력 예
사칙연산=A1+B1*2
합계·평균=SUM(D2:D20), =AVERAGE(D2:D20)
조건=IF(D2>0,"양수","음수")
다른 시트 참조='재무표'!D2-'재무표'!E2
반올림=ROUND(A1,2)

지원 함수: SUM, AVERAGE, MIN, MAX, COUNT, COUNTA, IF, ROUND, ROUNDUP, ROUNDDOWN, ABS, SQRT, POWER, MOD, CONCAT, LEN, AND, OR, NOT.

키보드

Enter/Tab 이동, F2 편집, F4 참조 고정 전환, Esc 편집·드래그 취소, Delete 내용 삭제, Ctrl/Cmd+C·V·X 복사·붙여넣기·잘라내기, Ctrl/Cmd+Z 실행 취소, Ctrl/Cmd+Y 다시 실행, Ctrl/Cmd+B 굵게.

Excel로 옮기기

Excel 호환은 모든 시트를 Excel에서 열 수 있는 Spreadsheet XML로 내보냅니다. 수식 결과값과 원본 수식이 함께 포함됩니다.
